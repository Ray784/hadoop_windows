Installing and Setting up Hadoop on Windows

1. Download required files:

  • Download hadoop: hadoop-2.8.0.tar.gz
  • Download java 64bit: java-64.zip
  • Download java 32bit: java-32.zip
  • Download winutils 64bit: winutils64-bin.zip
  • Download winutils 32bit: winutils32-bin.zip

2. Installation and setup:

  • Extract hadoop-2.8.0.tar.gz in c:\
  • Extract java-32.zip (or) java-64.zip in c:\
  • Extract winutils64-bin.zip (or) winutils32-bin.zip in c:\hadoop-2.8.0
  • set environment variables:
    • HADOOP_HOME = c:\hadoop-2.8.0
    • JAVA_HOME = c:\java
    • path = c:\java\bin; c:\hadoop-2.8.0\bin

3. Configuration:

    • create new folder data in c:\hadoop-2.8.0\
    • create new folder datanode in c:\hadoop-2.8.0\data
    • create new folder namenode in c:\hadoop-2.8.0\data

Go to c:\hadoop-2.8.0\etc\hadoop\
  • download and replace xml files:
    • 64-bit: xml-64.zip
    • 32-bit: xml-32.zip
    • (or)
  • find and edit:

  • 1. core-site.xml and replace with following code: 
    
<?xml version="1.0" encoding="UTF-8"?>
<?xml-stylesheet type="text/xsl" href="configuration.xsl"?>
<configuration>
	<property>
		<name>fs.defaultFS</name>
		<value>hdfs://localhost:9000</value>
	</property>
</configuration>
  • 2. mapred-site.xml and replace with following code:
    
<?xml version="1.0" encoding="UTF-8"?>
<?xml-stylesheet type="text/xsl" href="configuration.xsl"?>
<configuration>
	<property>
		<name>mapreduce.framework.name</name>
		<value>yarn</value>
	</property>
</configuration>
  • 3. hdfs-site.xml and replace with following code: 
    
<?xml version="1.0" encoding="UTF-8"?>
<?xml-stylesheet type="text/xsl" href="configuration.xsl"?>
<configuration>
	<property>
		<name>dfs.replication</name>
		<value>1</value>
	</property>
	<property>
		<name>dfs.namenode.name.dir</name>
		<value>C:\hadoop-2.8.0\data\namenode</value>
	</property>
	<property>
		<name>dfs.datanode.data.dir</name>
		<value>C:\hadoop-2.8.0\data\datanode</value>
	</property>
</configuration>
  • 4. yarn-site.xml and replace with following code: 
    
<?xml version="1.0"?>
<configuration>
   <property>
    	<name>yarn.nodemanager.aux-services</name>
    	<value>mapreduce_shuffle</value>
   </property>
   <property>
      	<name>yarn.nodemanager.auxservices.mapreduce.shuffle.class</name>  
	<value>org.apache.hadoop.mapred.ShuffleHandler</value>
   </property>
</configuration>
  • format the namenode by running the following command on cmd:
    hdfs namenode -format

4. Testing:

Go to c:\hadoop-2.8.0\sbin\
  • run start-all.cmd as administrator and make sure all of:
    • yarn-node manager
    • yarn-resource manager
    • Namenode
    • Datanode
    are running.
  • find hadoop GUI's at:
    • http://localhost:8080
    • http://localhost:50070

Running Mapreduce on hadoop to count words.

1. Starting hadoop hdfs:

Go to c:\hadoop-2.8.0\sbin\
  • run start-all.cmd as administrator and make sure all of:
    • yarn-node manager
    • yarn-resource manager
    • Namenode
    • Datanode
    are running.

2. Create input file:

  • create a file input_file.txt at c:\ and add some content.
  • (or)
  • download this file: input_file.txt and place it in c:\
for all of the following : open cmd in administrator mode

3. Create input directory in hdfs:

  • using the command:
    hadoop fs -mkdir /input_dir

4. Copy input file to hdfs:

  • using the command:
    hadoop fs -put c:\input_file.txt /input_dir

5. Check avalability of input file in hdfs:

  • using the command:
    hadoop fs -ls /input_dir/

6. Check content of input file:

  • using the command:
    hadoop fs -cat /input_dir/input_file.txt

7. Run map-reduce on input_file:

  • make sure the following file is available:
    c:\hadoop-2.8.0\share\hadoop\mapreduce\hadoop-mapreduce-examples-2.8.0.jar
  • run mapreduce using the command:
    hadoop jar c:\hadoop-2.8.0\share\hadoop\mapreduce\hadoop-mapreduce-examples-2.8.0.jar wordcount /input_dir /output_dir

8. Output:

  • After execution complete the output can be checked using the command:
    hadoop fs -cat /output_dir/*
